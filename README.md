# Nifty 100 Financial Intelligence Platform

A financial analytics platform built using Python, SQLite, Streamlit, and Plotly to analyze Nifty 100 companies.

## Features

- Financial Statement Analysis
- Ratio Analysis
- Stock Screener
- Peer Comparison
- Sector Analysis
- Company Reports
- Interactive Dashboard

## Tech Stack

- Python
- Pandas
- NumPy
- SQLite
- SQLAlchemy
- Streamlit
- Plotly
